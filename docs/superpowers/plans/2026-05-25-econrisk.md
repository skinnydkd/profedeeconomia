**Goal:** Build a hot-seat (1-4 humans + AI) Risk-style economics strategy game at `/juegos/econrisk/`, reusing the games framework.

**Architecture:** Astro page → full-screen Preact island. All game rules live in pure, deterministic TypeScript in `src/lib/games/econrisk/` (vitest, injectable RNG), same pattern as `src/lib/games/stonks/*`. UI in `src/components/games/econrisk/`. **Tech Stack:** Astro 5, Preact islands, TypeScript strict, Tailwind 4, vitest. No new dependencies.

**Reference spec:** `docs/superpowers/specs/2026-05-25-econrisk-design.md`. **Branch:** `feat/econrisk` (stacked on `feat/stonks`, which provides `GameShell` + `storage`).

## Conventions & constants
- Import alias for `src/lib`: `@/lib/...` (there is NO `@lib`). Components: `@components/...`, layout: `@layouts/...`.
- Match Preact patterns in `src/components/calculadoras/*.tsx` and `src/components/games/stonks/*.tsx` (`@jsxImportSource preact`, `class`, hooks from `preact/hooks`).
- Faction colors: `keynes`→`#1F6E6E`, `marx`→`#8C2F39`, `austrian`→`#A87A2A`, `neoclassic`→`#2E5E3A`.
- Rules: reinforcements `= Math.max(3, Math.floor(ownedCount/3)) + continentalBonus`. Combat: attacker rolls `min(3, attackingUnits-1)` dice, defender rolls `min(2, defendingUnits)`; sort each desc; compare the top pairs; on each pair the loser removes 1 unit; **ties go to the defender**. Victory: `ownedCount >= 18` OR only one faction alive OR at end of round 15 the faction with most territories wins.
- All randomness (events, dice, AI tie-breaks, initial distribution) goes through an injectable `rng: () => number` (default `Math.random`) so tests are deterministic.

- [ ] **Step 3: Implement `engine.ts`** — implement every export. Key rules (encode exactly):
  - `createInitialState(humanFactions, rng)`: shuffle territory order with `rng`, deal round-robin to the 4 factions, 1 unit each then distribute a few extra deterministically; set `factions[*].isHuman` from `humanFactions`, `alive=true`, `turnsSinceBonus=0`; `order` = FACTION_IDS; `current=0`; `round=1`; `phase='event'`; `peaceUntilRound=0`; `winner=null`.

  - `resolveAttack`: roll `min(3, from.units-1)` attacker dice and `min(2, to.units)` defender dice via `rng` (`Math.floor(rng()*6)+1`); **austrian defender: +1 to each defender die**; sort desc; compare top pairs; loser of each pair −1 unit; ties → defender keeps, attacker loses. If `to.units` hits 0 → attacker captures, moves `min(from.units-1, attackerDiceCount)` units in (keep ≥1 behind); set new owner; mark defender dead if 0 territories. **Marxist attacker**: if defender territory has exactly 1 unit, auto-capture without rolling.   - `endTurn`: reset per-turn flags (`neoclassicJumpUsed=false`); apply keynes power (`turnsSinceBonus++`; every 3 → +2 units + reset); advance `current` to next alive faction; if wrapped, `round++`; recompute `winner` via `checkVictory`; at round 15 end → winner = most territories.
  - `checkVictory`: 18+ → that faction; one alive → that faction; round>15 → most territories; else `null`.

## Task 8: MapView (SVG, clickable)

**Files:** Create `src/components/games/econrisk/MapView.tsx`

- [ ] **Step 1: Implement** a presentational SVG map: continent group backgrounds, adjacency lines (for each territory, draw a line to each higher-id neighbour to avoid duplicates), and a circle per territory filled with the owner's faction color showing the unit count; highlight `selectedId`; call `onSelect(id)` on click. Coordinates come from `map.ts` (`t.x`, `t.y`). Colors from `factions.ts`.

```tsx
/** @jsxImportSource preact */
import type { GameState } from '@/lib/games/econrisk/types';
import { TERRITORIES, byId } from '@/lib/games/econrisk/map';
import { factionMeta } from '@/lib/games/econrisk/factions';

export function MapView({ state, selectedId, onSelect }:
  { state: GameState; selectedId: string | null; onSelect: (id: string) => void }) {
  return (
    <svg viewBox="0 0 600 360" width="100%" role="img" aria-label="Mapa de territorios">
      <g stroke="#C9B79A" stroke-width="1.5">
        {TERRITORIES.flatMap((t) => t.adj.filter((n) => n > t.id).map((n) => (
          <line key={`${t.id}-${n}`} x1={t.x} y1={t.y} x2={byId[n].x} y2={byId[n].y} />
        )))}
      </g>
      <g font-family="JetBrains Mono, monospace" font-size="13" font-weight="700" text-anchor="middle">
        {TERRITORIES.map((t) => {
          const cell = state.territories[t.id];
          const col = factionMeta[cell.owner].color;
          return (
            <g key={t.id} class="er-node" onClick={() => onSelect(t.id)} style={{ cursor: 'pointer' }}>
              <circle cx={t.x} cy={t.y} r={16} fill={col}
                stroke={selectedId === t.id ? '#2A1F18' : 'none'} stroke-width={selectedId === t.id ? 3 : 0} />
              <text x={t.x} y={t.y + 5} fill="#fff">{cell.units}</text>
            </g>
          );
        })}
      </g>
    </svg>
  );
}
```
