Por qué este sitio existe

Llevamos años publicando material suelto repartido entre ocho índices distintos. Esto es lo mismo, pero ordenado como hace falta cuando uno entra a clase un lunes a las 8:15.

Todo lo que verás aquí parte del currículo básico estatal LOMLOE: Real Decreto 243/2022 para Bachillerato, Real Decreto 217/2022 para ESO. Cada comunidad autónoma añade su concreción, y conviene revisarla. Aquí no fingimos cubrir las diecisiete: cubrimos la base común, que ya es bastante.
